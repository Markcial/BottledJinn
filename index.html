About Bottled Jinn

Data manipulation framework built on top of Bottle: Python framework for the web environment Twitter Bootstrap 2.0 for the improved GUI and with Redis as the data persitence daemon.

Installation guide

Do a clone of the git repository 

$ git clone https://github.com/Markcial/BottledJinn.git
$ cd BottledJinn/
$ python main.py

Listening on http://localhost:8080/
Hit Ctrl-C to quit.
